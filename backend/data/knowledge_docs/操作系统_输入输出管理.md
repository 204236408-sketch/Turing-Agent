# 输入输出管理
subject: 操作系统
knowledge_point: 输入输出管理
keywords: 寻道时间,SSTF,SCAN,C-SCAN,磁盘调度

## 核心概念
磁盘调度算法：FCFS、SSTF、SCAN电梯、C-SCAN；SSTF易产生饥饿，SCAN双向扫描磁道。

## 408 寻道长度计算题
1. SSTF/SCAN执行顺序
2. SSTF饥饿问题

## 解题步骤
SSTF每次选最近磁道，远端请求长期等待

## 电梯算法扫描方向？
解答：单向到头再反向。

## 常见错误
SSTF无饥饿问题；C-SCAN来回扫描。
