# 树与二叉树
subject: 数据结构
knowledge_point: 树与二叉树
keywords: 满二叉树,完全二叉树,n0=n2+1

## 核心概念
二叉树度≤2，严格区分左右孩子；满二叉树每层填满；完全二叉树从上至下连续填充无空缺；公式n0=n2+1。

## 408 公式计算题
1. n0=n2+1
2. 满完全二叉树区分

## 解题步骤 叶子=度2节点+1

## 例题 n2=5，叶子？
解答：6

## 常见错误
二叉树左右无顺序；完全二叉树中间可空缺。
