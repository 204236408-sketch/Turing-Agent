# 计算机系统概述
subject: 计算机组成原理
knowledge_point: 计算机系统概述
keywords: 四代计算机,电子管,晶体管,集成电路,超大规模集成电路

## 核心概念
计算机发展划分为四代，依据电子元器件划分：电子管→晶体管→中小规模集成电路→超大规模集成电路；四代计算机运算速度、存储容量、体积、功耗逐步优化。

## 408 常见考法
1. 四代计算机元器件选择题
2. 四代发展顺序判断

## 解题步骤
一代电子管、二代晶体管、三代中小集成电路、四代超大规模集成电路

## 例题
第二代计算机采用硬件是？
解答：晶体管

## 常见错误
四代划分依据编程语言；第四代使用分立晶体管。
