# 存储系统
subject: 计算机组成原理
knowledge_point: 存储系统
keywords: 存储层次,Cache,主存,辅存,局部性原理,易失存储

## 核心概念
存储层次：寄存器→Cache→主存→辅存；速度递减、容量递增、单位成本降低；局部性原理是分层存储设计基础；存储介质分易失、非易失。

## 408 简答选择
1. 存储层次速度容量排序
2. 局部性原理作用

## 解题步骤
速度：寄存器>Cache>主存>硬盘

## 例题
访问速度最快存储？
解答：CPU内部寄存器

## 常见错误
硬盘速度高于内存；分层和局部性无关。
