# 传输层
subject: 计算机网络
knowledge_point: 传输层
keywords: 三次握手,四次挥手,TCP首部,序号,确认ACK

## 核心概念
TCP面向连接可靠传输；三次握手同步初始序号，四次挥手双向断开；首部20~60字节，包含序号、ACK确认、窗口字段。

## 高频简答计算题
1. 三次握手四次挥手流程
2. 拥塞控制四阶段切换

## 解题步骤
握手同步序号；慢启动指数增长，3个冗余ACK触发快重传

## 例题：SYN报文携带数据？
解答：不能

## 常见错误
SYN报文携带业务数据；挥手仅三次报文。
