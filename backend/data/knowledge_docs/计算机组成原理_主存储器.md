# 存储系统
subject: 计算机组成原理
knowledge_point: 存储系统
keywords: MAR,MDR,位扩展,字扩展,RAM,ROM

## 核心概念
主存由存储体、地址译码器、MAR、MDR构成；存储芯片扩展分为位扩展、字扩展、字位同时扩展；RAM随机读写，ROM只读不可写。

## 408 芯片扩展计算题
1. 位扩展、字扩展计算芯片数量
2. RAM/ROM区分

## 解题步骤
芯片总数 = 字数倍数 × 位数倍数

## 例题
16K×8 使用2K×4芯片，需要几片？
解答：16片

## 常见错误
位扩展增加地址线；ROM支持随机写入。
