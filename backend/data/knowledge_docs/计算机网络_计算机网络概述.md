# 计算机网络体系结构
subject: 计算机网络
knowledge_point: 计算机网络体系结构
keywords: 自治计算机,数据通信,资源共享,电路交换,分组交换,时延,带宽,RTT

## 核心概念
计算机网络是由多台自治计算机通过通信设备互连形成的系统；核心两大功能：数据通信、资源共享；交换方式分为电路交换、报文交换、分组交换；性能指标包括带宽、时延、吞吐量、往返时间RTT。

## 408 常见考法
1. LAN/MAN/WAN范围区分选择题
2. 四类时延计算题
3. 三种交换方式对比简答

## 解题步骤
发送时延=分组长度/链路速率；传播时延=距离/信号速度
电路交换独占链路；分组存储转发、链路共享

## 例题
1000bit分组，100Mbps链路，发送时延？
解答：1000 / 100e6 = 10μs

## 常见错误
互连等同于互联；分组交换无存储转发；带宽等同于传播速率。

# 计算机网络体系结构
subject: 计算机网络
knowledge_point: 计算机网络体系结构
keywords: LAN,MAN,WAN,端系统,路由器,主机,局域网广域网

## 核心概念
计算机网络按覆盖范围划分：局域网LAN、城域网MAN、广域网WAN；局域网传输距离短、速率高；广域网依靠路由器跨地域转发；主机是端系统，路由器是中间交换设备。

## 408 常见考法
1. LAN/MAN/WAN范围区分选择题
2. 四类时延计算题
3. 三种交换方式对比简答

## 解题步骤
发送时延=分组长度/链路速率；传播时延=距离/信号速度
电路交换独占链路；分组存储转发、链路共享

## 例题
1000bit分组，100Mbps链路，发送时延？
解答：1000 / 100e6 = 10μs

## 常见错误
MAN规模大于WAN；集线器可以跨网段互联主机。
