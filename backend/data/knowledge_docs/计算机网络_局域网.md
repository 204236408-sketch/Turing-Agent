# 数据链路层
subject: 计算机网络
knowledge_point: 数据链路层
keywords: 以太网,MAC地址,最小帧长,802.3,半双工冲突

## 核心概念
以太网标准802.3，48位MAC硬件地址；最小帧64字节，最大1518字节；共享总线半双工采用CSMA/CD冲突检测。

## 408 计算判断
1. 以太网最小帧长推导
2. MAC地址作用

## 解题步骤
最小帧长=往返时延×传输速率，保证冲突被检测

## 例题
MAC地址可以跨路由器转发？
解答：不可以

## 常见错误
MAC地址可跨路由器路由；最小帧不含首部。
