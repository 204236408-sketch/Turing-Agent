# 应用层
subject: 计算机网络
knowledge_point: 应用层
keywords: HTTP,HTTPS,TLS,GET,POST,持久连接

## 核心概念
HTTP 80端口无状态；HTTPS 443 TLS加密；GET参数明文URL，POST放请求体；HTTP1.1持久连接复用TCP。

## 408 HTTP/HTTPS概念
1. GET POST区别
2. 无状态特性

## 解题步骤
HTTP无状态，依靠Cookie维持会话；HTTPS全链路TLS加密

## 例题：HTTP默认端口？
解答：80

## 常见错误
HTTP保存会话状态；HTTPS仅加密URL路径。
