# 绪论
subject: 数据结构
knowledge_point: 绪论
keywords: 算法五大特性,有穷性,程序与算法区别

## 核心概念
算法是有限步骤指令序列，五大特性：有穷、确定、可行、输入、输出；程序不满足有穷性，不能等同于算法。

## 408 常见考法
1. 循环代码复杂度计算
2. 空间递归计算
3. 五大特性多选

## 解题步骤
复杂度只保留最高阶；递归空间=深度

## 例题
for(i=0;i<n;i++) sum++; 复杂度？
解答：O(n)

## 常见错误
算法=程序；无限循环程序属于算法。
