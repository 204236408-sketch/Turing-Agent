# 数据链路层
subject: 计算机网络
knowledge_point: 数据链路层
keywords: CSMA/CD,CSMA-CA,ALOHA,令牌环,介质访问MAC

## 核心概念
三类MAC方式：信道划分(FDM/TDM)、轮询(令牌环)、随机访问(ALOHA/CSMA/CD/CSMA-CA)；CSMA/CD用于有线以太网，CSMA-CA无线局域网。

## 408 区分简答
1. CSMA/CD、CSMA-CA适用场景
2. 随机访问协议优缺点

## 解题步骤
有线以太网CSMA/CD；无线冲突避免CSMA-CA

## 例题
全双工以太网需要冲突检测？
解答：不需要

## 常见错误
全双工以太网使用CSMA/CD；令牌无冲突无需等待。
