# 计算机系统概述
subject: 操作系统
knowledge_point: 计算机系统概述
keywords: 用户态,内核态,系统调用,中断,异常,态切换

## 核心概念
CPU双模式：用户态(目态)、内核态(管态)；普通函数停留在用户态，系统调用是唯一合法切换入口；中断、异常触发态切换。

## 408 常见考法
1. 用户态切内核唯一途径选择
2. 系统调用vs普通函数区分

## 解题步骤
普通函数全程用户态；系统调用触发内核态切换

## 例题
判断：普通函数调用可以进入内核态。
解答：错误。

## 常见错误
普通函数可以切换内核态；应用程序直接访问内核寄存器。
