# 传输层
subject: 计算机网络
knowledge_point: 传输层
keywords: 端到端,端口,复用分用,TCP,UDP,进程通信

## 核心概念
传输层实现端到端进程通信；端口号区分主机内不同应用；分为无连接UDP、面向连接TCP；复用/分用功能区分上层应用。

## 408 判断
1. 端口作用区分进程
2. TCP/UDP基础差异

## 解题步骤
IP区分主机，端口区分主机内部应用进程

## 例题：IP地址唯一标识应用？
解答：错误

## 常见错误
IP地址区分进程；网络层实现端到端通信。
