# 进程与线程
subject: 操作系统
knowledge_point: 进程与线程
keywords: 临界资源,临界区,同步,互斥,并发竞争

## 核心概念
临界资源同一时刻仅允许一个进程独占访问；临界区是操作临界资源的代码段；互斥解决竞争，同步约束进程执行先后顺序。

## 408 大题必考PV操作
1. 互斥同步信号量初值设置
2. PV顺序错误后果分析

## 解题步骤
互斥S=1；同步按缓冲区/资源数设N/0

## 例题 生产者消费者空缓冲初值？
解答：缓冲区大小N。

## 常见错误
同步等价于互斥；临界区指硬件设备。

keywords: 信号量,P操作,V操作,管程,互斥锁

## 核心概念
同步互斥实现方案：硬件锁、信号量PV操作、管程；互斥信号量初值为1，同步信号量根据等待资源数量设0/N。

## 408 大题必考PV操作
1. 互斥同步信号量初值设置
2. PV顺序错误后果分析

## 解题步骤
互斥S=1；同步按缓冲区/资源数设N/0

## 例题 生产者消费者空缓冲初值？
解答：缓冲区大小N。

## 常见错误
V操作会阻塞进程；互斥信号量初值设0。
