# 网络层
subject: 计算机网络
knowledge_point: 网络层
keywords: 异构互连,路由,分组转发,拥塞控制,IP服务

## 核心概念
网络层核心：异构网络互连、路由选择、分组转发、拥塞控制；屏蔽下层链路差异，向上提供统一IP数据报服务。

## 408 多选简答
1. 网络层四大核心功能
2. 跨网互联原理

## 解题步骤
网络层提供主机到主机无连接分组服务

## 例题
端到可靠传输属于哪一层？
解答：传输层TCP

## 常见错误
网络层实现端到端可靠传输；路由仅静态配置。
