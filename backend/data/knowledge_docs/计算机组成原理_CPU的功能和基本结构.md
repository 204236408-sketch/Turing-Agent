# 中央处理器
subject: 计算机组成原理
knowledge_point: 中央处理器
keywords: PC,IR,PSW,通用寄存器,运算器,控制器

## 核心概念
CPU=运算器+控制器；内部寄存器：PC、IR、MAR、MDR、PSW、通用寄存器；PC存下一条指令地址，IR存放当前指令。

## 408 判断选择
1. PC、IR寄存器作用
2. CPU组成部件

## 解题步骤
PC存下一条指令地址

## 例题
IR存放什么内容？
解答：当前整条指令

## 常见错误
PC存储当前指令；IR保存操作数。
