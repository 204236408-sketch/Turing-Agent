# 查找
subject: 数据结构
knowledge_point: 查找
keywords: 顺序查找,二分查找,判定树

## 核心概念
顺序查找适配无序/链表，ASL=(n+1)/2；二分仅有序连续顺序表可用，mid=low+(high-low)//2防溢出。

## 408 ASL计算
1. 二分判定树高度
2. 适用存储结构

## 例题 长度7有序表二分成功ASL？
解答：可分步计算

## 常见错误
有序链表可二分；mid直接相加溢出。
