# 线性表
subject: 数据结构
knowledge_point: 线性表
keywords: 线性表,有序序列,前驱后继,空线性表
高频考点: True

## 核心概念
线性表是零个或多个同类型数据元素的有序线性序列；第一个元素无前驱，最后一个无后继，其余元素唯一前驱唯一后继；空线性表长度为0。
