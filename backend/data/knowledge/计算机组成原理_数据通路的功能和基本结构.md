# 中央处理器
subject: 计算机组成原理
knowledge_point: 中央处理器
keywords: 单总线数据通路,专用通路,总线竞争,并行传输
高频考点: True

## 核心概念
数据通路分为单总线结构、专用分离通路；单总线所有寄存器分时共享一条内部总线，硬件简单传输冲突；专用通路并行传输速度快硬件成本高。

## 例题
见本节对应408经典真题解析
