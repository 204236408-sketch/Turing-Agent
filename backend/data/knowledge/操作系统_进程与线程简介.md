# 进程与线程
subject: 操作系统
knowledge_point: 进程与线程
keywords: 进程,程序,动态实例,并发性,独立地址空间,异步执行
高频考点: True

## 核心概念
进程是具有独立资源、独立调度权的程序运行实例，具备动态性、并发性、独立性、异步性四大特征；程序是静态代码文件，一个程序可生成多个进程。
