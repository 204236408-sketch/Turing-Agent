# 文件管理
subject: 操作系统
knowledge_point: 文件管理
keywords: 硬链接,软链接,inode引用计数,跨分区限制
高频考点: True

## 核心概念
硬链接指向同一inode，引用计数累加，不可跨分区；软链接存储文件路径，独立inode，原文件删除链接失效。
