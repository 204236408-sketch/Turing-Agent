# 计算机系统概述
subject: 操作系统
knowledge_point: 计算机系统概述
keywords: 单内核,微内核,混合内核,内核态,进程通信,设备驱动
高频考点: True

## 核心概念
操作系统内核分为单内核、微内核、混合内核。单内核全部功能运行内核态，调用高效；微内核仅调度、通信在内核，驱动用户态，稳定性强；混合内核融合两者优势。
