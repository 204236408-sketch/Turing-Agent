# 排序
subject: 数据结构
knowledge_point: 排序
keywords: 稳定排序,内部排序,外部排序
高频考点: True

## 核心概念
稳定排序相等元素相对位置不变；不稳定会交换相等元素位置；内部排序全部数据载入内存，外部排序依赖磁盘。

## 408 常见考法
1. 概念辨析选择题
2. 计算/简答类真题

## 常见错误
