# ChromaDB Collections

| Collection | 内容 | 用途 |
|---|---|---|
| `knowledge_base_408` | 408 知识点、考点和题目解析 | 问答、出题、论坛分析 |
| `conversation_summary` | 会话摘要 | 连续问答与长期上下文 |
| `user_memory_vector` | 用户长期记忆摘要 | 个性化推荐与报告 |
| `mastery_summary` | 掌握状态摘要 | 首页推荐与智能出题 |
| `mistake_summary` | 错题和错因摘要 | 相似错题检索 |
| `forum_post_summary` | 论坛帖子摘要 | 相似帖子和论坛助手 |
| `forum_ai_answer_summary` | AI 回答摘要 | 帖子追问 |
| `report_summary` | 历史报告摘要 | 长期报告追踪 |

统一 metadata：`user_id`、`subject`、`knowledge_point`、`final_status`、
`source_type`、`source_id`、`memory_type`、`post_id`、`create_time`。
