项目故事

408 考研涵盖四门专业课、数百个知识点，考生普遍面临知识体系庞杂、错题无法系统消化、复习缺乏针对性的困境。传统刷题工具只能记录对错，无法真正理解"为什么错"以及"接下来该学什么"。

本项目以多 Agent 协同架构驱动完整学习闭环——问答→练习→批改→归因→记忆→再推荐，每一次交互都在构建你的个人学习画像。AI 不仅给出答案，更追溯知识源头、分析错误模式、动态调整复习计划，让备考从盲目刷题升级为精准进化。

技术架构

FastAPI LangGraph OpenAI API ChromaDB SQLite Vanilla JS Docker Nginx

后端采用 Python FastAPI 构建，基于 LangGraph 实现多 Agent 工作流；前端保持轻量级原生 JavaScript，支持三种主题切换。
