项目缘起

《重生之我是图灵》源于一个简单而浪漫的想法：如果艾伦·图灵本人来备考计算机 408，他会如何整理知识、如何思考问题、如何设计复习策略？

我们将这一想象融入系统设计，以图灵的计算思维为灵魂，以现代大语言模型为引擎，打造了一个真正"懂 408"的 AI 辅导系统。

技术架构

FastAPI LangGraph OpenAI API ChromaDB SQLite Vanilla JS Docker Nginx

后端采用 Python FastAPI 构建，基于 LangGraph 实现多 Agent 工作流；前端保持轻量级原生 JavaScript，支持三种主题切换。
