# 项目答辩稿

各位老师好，我们的项目是“重生之我是图灵：面向考研 408 的 Web 智能辅导 Agent 系统”。

项目面向 408 复习中的三个痛点：知识点多、错因难沉淀、练习缺少个性化。系统用 Web 工作台承载学习流程，用 FastAPI 提供后端服务，用 Agent 编排问答、出题、批改、错题分析、论坛 AI 和学习报告。

我们的核心不是做一个普通聊天助手，而是形成学习闭环：用户问问题，系统识别知识点并读取长期记忆；用户做题后，系统批改并更新掌握状态；用户确认错因后，错题会写入长期记忆；后续出题和报告都会使用这些记忆。

技术上，前端采用 HTML、CSS、JavaScript，保证轻量稳定；后端采用 FastAPI、SQLAlchemy，默认 SQLite 便于演示，后续可切换 MySQL。系统保留 ChromaDB、PaddleOCR 和大模型接口位置，开发版先用可运行 Mock 保证完整闭环。

最终成果包括完整前端页面、后端接口、数据库设计、统一报错机制、部署说明和五人六天分工方案。
