# 数据库设计

## 核心域

- 用户：`user`、`user_profile`
- 知识：`subject`、`knowledge_point`
- 出题：`question_generation_session`、`question`
- 答题：`answer_record`、`question_interaction_log`
- 掌握状态：`knowledge_mastery`、`mastery_event`
- 错题：`mistake`
- 问答记忆：`qa_record`、`conversation`、`conversation_message`、`user_memory`
- 视频：`video_resource`、`video_view_record`
- 论坛：分类、帖子、评论、点赞、AI 回答、AI 追问、打卡
- 报告：`report`

正式字段和约束以 `database/migrations/` 为准，`backend/models.py` 必须保持同步。
