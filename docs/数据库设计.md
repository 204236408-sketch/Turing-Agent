# 数据库设计

核心表：

- `user`：账号、昵称、密码哈希、状态。
- `user_profile`：目标考试、每日学习时长、学习阶段。
- `subject` / `knowledge_point`：408 四科和知识点层级。
- `question_generation_session`：每次出题批次。
- `question`：题干、选项、答案、解析、提示。
- `answer_record`：用户答案、正误、反馈。
- `knowledge_mastery`：掌握状态、答题统计、薄弱分。
- `mistake`：错题、错因、建议、来源。
- `user_memory`：长期学习画像和证据。
- `conversation` / `conversation_message`：长对话。
- `forum_category` / `forum_post` / `forum_comment`：论坛。
- `video_resource`：题目内嵌视频推荐。
- `report`：学习报告。

完整 MySQL 建表参考见 `database/schema.sql`。开发版由 SQLAlchemy 自动创建同构表。
