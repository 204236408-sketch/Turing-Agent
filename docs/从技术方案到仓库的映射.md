# 从技术方案到仓库的映射

| 技术方案内容 | 仓库位置 |
|---|---|
| version-b 前端 | `frontend/` |
| FastAPI 入口与配置 | `backend/main.py`、`config.py` |
| JWT 与数据库依赖 | `backend/auth.py`、`database.py`、`dependencies.py` |
| 业务接口 | `backend/routers/` |
| 核心服务 | `backend/services/` |
| LangGraph Agent | `backend/agents/` |
| Prompt 模板 | `backend/prompts/` |
| ChromaDB | `backend/vector_store/` |
| PaddleOCR | `backend/services/ocr_service.py` |
| 初始化数据 | `backend/data/` |
| 运维脚本 | `backend/scripts/` |
| MySQL SQL | `database/` |
| Nginx 与进程部署 | `deploy/` |
| 自动化验收 | `tests/` |

## 与旧结构的变化

- `apps/web` 被替换为 `frontend/`，不再规划 Vue3。
- `apps/api`、`services/ai`、`services/agent` 合并进 `backend/`。
- ChromaDB 持久化目录迁移到 `backend/vector_store/chroma/`。
- OCR 上传、报告和日志迁移到 `backend/uploads/` 与 `backend/outputs/`。
- 最终 `version-b` 直接成为可部署前端，A/C 方案进入文档归档。
