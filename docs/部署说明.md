# 部署说明

## 本地演示

```powershell
python -m pip install -r backend\requirements.txt
cd backend
python -m uvicorn main:app --reload --host 127.0.0.1 --port 8000
```

## MySQL 切换

1. 创建数据库 `yantu408`。
2. 复制 `backend/.env.example` 为 `backend/.env`。
3. 配置：

```text
MYSQL_URL=mysql+pymysql://root:password@127.0.0.1:3306/yantu408?charset=utf8mb4
```

4. 安装 PyMySQL：

```powershell
python -m pip install pymysql
```

5. 重启后端。

## Nginx

参考 `deploy/nginx.conf`，把 `/api` 代理到后端，静态页面指向项目 `frontend` 目录。
