# 演示说明

推荐演示路线：

1. 打开登录页，使用演示账号进入系统。
2. 首页展示倒计时、今日推荐、知识图谱和掌握状态。
3. 进入知识问答，问“LRU 和 FIFO 有什么区别”，展示 Agent 步骤。
4. 进入智能出题，使用智能推荐生成 3 道题。
5. 提交错误答案，选择错因，展示错题和长期记忆写入。
6. 进入论坛，打开 AI 小助手回答。
7. 生成学习报告，展示下一轮复习计划。

答辩重点：

- 前端延续 version-b，无框架，低成本稳定落地。
- 后端分层清楚，接口统一报错。
- Agent 不只是聊天，而是贯穿问答、出题、批改、错题、论坛和报告。
- 记忆闭环能把一次错误变成后续推荐依据。
