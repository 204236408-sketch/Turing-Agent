# 计算机系统概述
subject: 计算机组成原理
knowledge_point: 计算机系统概述
keywords: 五大硬件,CPU,运算器,控制器,系统软件,应用软件

## 核心概念
计算机系统分为硬件层与软件层；硬件五大部件：运算器、控制器、存储器、输入设备、输出设备；运算器+控制器合称CPU；软件分系统软件、应用软件。

## 408 常见考法
1. 五大硬件部件多选
2. CPU组成判断题

## 解题步骤
CPU = 运算器 + 控制器，不含存储器

## 例题
判断题：CPU包含主存芯片。
解答：错误

## 常见错误
CPU包含主存；应用软件管理硬件资源。
