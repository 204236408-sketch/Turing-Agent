# 数据链路层
subject: 计算机网络
knowledge_point: 数据链路层
keywords: 成帧,透明传输,差错检测,流量控制,链路管理

## 核心概念
数据链路层五大核心功能：封装成帧、透明传输、差错检测、流量控制、链路管理；向上屏蔽底层物理比特传输细节。

## 408 简答多选
1. 链路层五大功能记忆
2. 各层功能区分

## 解题步骤
链路层只处理链路拥塞，全局拥塞属于网络层

## 例题
下列不属于数据链路层功能？A成帧 B路由 C CRC
解答：B路由（网络层）

## 常见错误
链路层实现拥塞控制；透明传输删除控制字符。
