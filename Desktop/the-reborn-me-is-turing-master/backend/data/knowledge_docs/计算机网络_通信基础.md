# 物理层
subject: 计算机网络
knowledge_point: 物理层
keywords: 码元,奈奎斯特,香农,带宽,信噪比,比特率,波特率

## 核心概念
通信分为模拟通信、数字通信；码元是离散信号基本单元；奈奎斯特定理：无噪声信道最大码元速率2W Baud；香农公式计算带噪声信道极限比特率C=Wlog2(1+S/N)。

## 408 高频计算题
1. 奈奎斯特最大码元速率
2. 香农信道极限速率
3. 编码调制区分判断

## 解题步骤
奈奎斯特：2*带宽 Baud；香农C=Wlog2(1+S/N)

## 例题
无噪声4kHz信道最大码元速率？
解答：8000 Baud

## 常见错误
奈奎斯特计算比特率；香农不限制码元数量。

3. 编码调制区分判断

奈奎斯特：2*带宽 Baud；香农C=Wlog2(1+S/N)

## 例题
无噪声4kHz信道最大码元速率？
解答：8000 Baud

## 常见错误
调制属于数字转数字；曼彻斯特无时钟同步。
