# 数据链路层
subject: 计算机网络
knowledge_point: 数据链路层
keywords: 字符填充,比特填充,违规编码,PPP成帧,帧边界

## 核心概念
四种成帧方法：字符计数、字符填充、比特填充、违规编码；PPP协议使用字节填充，以太网采用曼彻斯特违规编码定帧边界。

## 408 判断简答
1. 四种成帧方案优缺点
2. PPP字节填充规则

## 解题步骤
比特填充遇到连续5个1后置0；字符填充转义标志字节

## 例题
PPP帧透明传输依靠什么？
解答：字节填充法

## 常见错误
字符计数可靠无差错；比特填充仅填充0比特。
