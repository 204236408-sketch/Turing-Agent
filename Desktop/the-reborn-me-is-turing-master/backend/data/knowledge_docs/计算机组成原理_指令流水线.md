# 中央处理器
subject: 计算机组成原理
knowledge_point: 中央处理器
keywords: 流水线,结构冒险,数据冒险,控制冒险,旁路转发

## 核心概念
流水线分段：取指、译码、执行、访存、写回；三类冒险：结构、数据、控制；旁路转发解决数据冒险，分支预测缓解控制冒险。

## 408 冒险判断简答
1. 三类冒险区分
2. 旁路转发原理

## 解题步骤
数据冒险可通过旁路转发消除阻塞

## 例题
连续读写同一寄存器是什么冒险？
解答：数据冒险

## 常见错误
流水线一定提速；数据冒险只能阻塞等待。
