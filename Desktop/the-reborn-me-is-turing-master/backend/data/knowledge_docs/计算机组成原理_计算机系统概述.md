# 计算机系统概述
subject: 计算机组成原理
knowledge_point: 计算机系统概述
keywords: 四代计算机,电子管,晶体管,集成电路,超大规模集成电路

## 核心概念
计算机发展划分为四代，依据电子元器件划分：电子管→晶体管→中小规模集成电路→超大规模集成电路；四代计算机运算速度、存储容量、体积、功耗逐步优化。

## 408 常见考法
1. 四代计算机元器件选择题
2. 四代发展顺序判断

## 解题步骤
一代电子管、二代晶体管、三代中小集成电路、四代超大规模集成电路

## 例题
第二代计算机采用硬件是？
解答：晶体管

## 常见错误
四代划分依据编程语言；第四代使用分立晶体管。

# 计算机系统概述
subject: 计算机组成原理
knowledge_point: 计算机系统概述
keywords: 五大硬件,CPU,运算器,控制器,系统软件,应用软件

## 核心概念
计算机系统分为硬件层与软件层；硬件五大部件：运算器、控制器、存储器、输入设备、输出设备；运算器+控制器合称CPU；软件分系统软件、应用软件。

## 408 常见考法
1. 五大硬件部件多选
2. CPU组成判断题

## 解题步骤
CPU = 运算器 + 控制器，不含存储器

## 例题
判断题：CPU包含主存芯片。
解答：错误

## 常见错误
CPU包含主存；应用软件管理硬件资源。

# 计算机系统概述
subject: 计算机组成原理
knowledge_point: 计算机系统概述
keywords: 主频,时钟周期,CPI,CPU执行时间,MIPS,FLOPS

## 核心概念
核心性能指标：主频、时钟周期、CPI、CPU执行时间、MIPS、FLOPS；CPU执行时间 = 指令条数 × CPI / 主频；CPI为单条指令平均时钟周期数。

## 408 高频计算题
1. CPU执行时间换算
2. CPI、主频换算

## 解题步骤
执行时间 = 指令总数 × CPI / 主频

## 例题
主频3GHz，5000指令，CPI=1，求执行时间？
解答：5000 / 3e9

## 常见错误
主频越高性能一定更强；MIPS可跨架构对比性能。
