# 数据的表示和运算
subject: 计算机组成原理
knowledge_point: 数据的表示和运算
keywords: 原码,反码,补码,Booth乘法,恢复余数,溢出判断

## 核心概念
定点数四种编码：原码、反码、补码、移码；补码加减统一公式；原码一位乘、Booth补码乘法；除法分为恢复余数、不恢复余数法；双符号位溢出判断。

## 408 大题高频
1. 原/反/补码转换
2. Booth乘法、溢出判断

## 解题步骤
负数补码 = 反码 + 1；双符号位不同代表溢出

## 例题
8位 [-8]补码？
解答：11111000

## 常见错误
补码存在正负零；减法无需加负数补码。
