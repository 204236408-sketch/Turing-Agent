# 接口文档

统一成功响应：

```json
{"ok": true, "message": "success", "data": {}}
```

统一失败响应：

```json
{"ok": false, "error": {"code": "VALIDATION_ERROR", "message": "参数不完整或格式错误", "detail": [], "request_id": "..."}}
```

- `GET /api/auth/me`
- `GET /api/home/overview`
- `GET /api/knowledge/graph`
- `GET /api/mastery/list`
- `POST /api/qa/chat`
- `POST /api/questions/generate`
